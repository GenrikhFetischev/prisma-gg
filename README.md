# PSINA üêæ
Blazzzzingly fast graphql schema generator from prisma file

All types from Prisma which are not represented in GraphQL scalars list (Int, Float, String, Boolean, and ID) will be added as `scalar` into graphql prisma file, for instance that:
```prisma
enum UserRole {
    user
    admin
}

model User {
  id            BigInt    @id @map("user_id")
  externalId    String    @unique @map("external_id")
  name          String
  role          UserRole?
  description   String?
  pic           String?
  createdAt     DateTime
  tags          Strng[]
  
  @@index([externalId])
  @@map("user")
}
```

turns into the following:

```graphql
# This file is generated by PSINA üêæ. Please, do not change it directly
# PSINA package: https://crates.io/crates/psina ‚≠êÔ∏è

scalar Strng
scalar BigInt
scalar DateTime


enum UserRole {
    user
    admin
}

type User {
    id: BigInt!
    externalId: String!
    name: String!
    role: UserRole
    description: String
    pic: String
    createdAt: DateTime!
    tags: [Strng!]!
}
```
## CLI
```shell
psina --prisma-file $PWD/schema.prisma --output-gql $PWD/schema.graphql
```
